# COVID-19 Vaccination Portal

## 📌 Project Overview
This project is a Python-based portal that checks the vaccination status of users using their User ID.

## 📊 Dataset
The dataset contains:
- Userid
- Name
- 1st Dose Status
- 1st Dose Date
- 2nd Dose Status
- 2nd Dose Date

## 🚀 Features
- Check vaccination status using User ID
- Displays personalized messages:
  - Fully vaccinated
  - First dose pending
  - Second dose reminder
- Calculates next dose date (after 90 days)

## 🛠️ Technologies Used
- Python
- Pandas
- Datetime

## ▶️ How to Run
1. Open the Jupyter Notebook
2. Run all cells
3. Enter User ID when prompted

## 📌 Example Output
