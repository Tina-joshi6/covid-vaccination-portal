# COVID-19 Vaccination Portal

## 📌 Project Overview

This project is a Python-based system that checks COVID-19 vaccination status using a dataset.

## 📊 Features

* Check vaccination status by User ID
* Displays personalized messages
* Calculates next dose date (after 90 days)

## 🛠️ Technologies Used

* Python
* Pandas
* Datetime

## ▶️ How to Run

1. Open the notebook file
2. Run all cells
3. Enter User ID

## 📎 Files

* vaccine_data.xlsx
* vaccination_portal.ipynb
